LogDog is...

다음과 같은 오픈소스들을 바탕으로 하여 만들어졌습니다.

Use Open Source and Licence

  • Jersey:우리는 최대한 Restful한 웹을 구성하기 위하여 Jersey라는 Web 프레임워크를 이용하였다.
  • MicroLog4Android: 안드로이드 용 로그 라이브러리로 우리는 이 오픈소스 라이브러리를 리팩토링하여 이 프로젝트의 로그 기능을 제작하였다. MicroLog4Android는 Apache License 2.0 라이센스이다.
  • HighCharts JS: 우리는 웹 상에 그래프를 그리기 위하여 HighCharts를 이용하였다. 이 라이브러리는 CC BY 3.0이다.
  • JqGrid: 우리는 웹 상에 테이블응 그리기 위하여 jqGrid 이용하였다. 이 라이브러리는 MIT license 이다.
  • bootstrap: 우리는 웹페이지 구성을 위하여 BootStap을 이용하였다. 이 오픈소스 라이브러리는 Apache License 2.0 라이센스이다.
  • Gson: 우리는 Json데이터와 자바 객체사이에 연동해주기 위하여 Gson을 이용하였다. 이 오픈소스 라이브러리는 Apache License 2.0 라이센스이다.
  • Simple framework: 우리는 XML데이터와 자바 객체사이에 연동해주기 위하여 Simple framework를 이용하였다. 이 오픈소스 라이브러리는 Apache License 2.0 라이센스이다.

추가로 이 Web Page는 Google App Engine for Java 기반으로 제작되었으며, 이 프로젝트의 자체적인 라이센스는 MIT 라이센스를 채택하고 있다.
